AECHO V2 Help - Menu > File

<Return to Help's Home Page>

The menu choice menu >  File menu provides five sub-menu choices: each supports a single-character "Alt"-key accelerator, and has a defined "shortcut". These shortcuts follow common Windows application conventions for the analogous functions seen in most applications.

Sub-Menu Choice
 Shortcut
 Purpose / Description
Open Hauptwerk Organ...
 		Ctrl+O
 Selects and loads a Hauptwerk ODF file, placing its contents into the ODF Display Area, and positioning to the DisplayPage Section. AECHO presents the operating-system's standard file-open dialog box to choose an ODF file, allowing the user to freely navigate their computer's file system to locate their file. This dialog presents the options to "Open" a selected file, or to "Cancel" the open.

The Open Hauptwerk Organ... menu choice is always enabled for use.

Only one ODF at a time may be loaded into AECHO, and only one instance of AECHO can be run at a time.

The first time it is run, AECHO determines an initial ODF directory from the file "initial.txt" located in AECHO's Data Directory. However, Windows retains the most recent directory from which an ODF was successfully loaded, and makes that the default ODF directory for subsequent Opens. This information is retained between uses of AECHO.

If an ODF is presently loaded, but the user decides to "Cancel" out of opening a new ODF, the previous ODF remains loaded. If a currently loaded ODF has unsaved changes, AECHO will notify the user, and give them the choice to "Save As..." the current ODF before proceeding with opening the new one. At the bottom of the main AECHO window, on the Status-Bar, there is a colored indicator that shows if there is an ODF presently loaded, and if so, does it have unsaved changes. This indicator is colored orange when no ODF is loaded, green when an ODF is loaded but has no unsaved changes, and red if there are unsaved changes.
Close ODF...
 		Ctrl+W
 Closes the currently loaded ODF, and clears the related display areas and data.

The Close ODF... menu choice is only enabled while an ODF is loaded.

If the presently loaded ODF has unsaved changes, AECHO will offer to save them before closing the ODF.
Save As...
 		Ctrl+Alt+S
 Saves the presently loaded ODF to the computer's file system. AECHO uses the operating-system's standard save-file dialog to allow the user to navigate to a desired directory and name their output file. The user may either click on an existing ODF file, or manually enter the filename to be used. The system will warn the user if they are attempting to overwrite an existing ODF file, but will permit it. The user may choose to either "Save" the ODF, or to "Cancel" the operation.

Note - it is highly recommended to always save a modified ODF under a new name, rather than overwriting an existing file.

The Save As... menu choice is only enabled while an ODF is loaded.

When an ODF is saved, AECHO resets the "unsaved changes" indicator on the Status-Bar, returning the indicator to green.
Print Descriptive Text...
 		Ctrl+P
 Prints the contents of the Descriptive Text Area (used to display Section Descriptions, Section locations, and fall-back Help text). Print dispatches the operating-system's print dialog, allowing the user to choose a destination printer, which can include physical printers, PDF printers, direct-to-file, or any other printer known to the system. Print output retains all the formatting of its source text, additionally adding a meaningful header at the top of the first output page. The dialog presents options to "Print" or to "Cancel".

The Print Descriptive Text... menu choice is only enabled while there is content displayed in the Descriptive Text Area.

Print supports modifying printer preferences and printing multiple copies, but only permits the Print-Range to be "All" - it does not directly support printing only selected text, or a page range.
Exit
 		Alt+F4
 Terminates AECHO.

The Exit menu choice is always enabled.

If there is a loaded ODF with unsaved changes, AECHO will offer to save the ODF before terminating.
